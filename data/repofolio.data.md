# Public Repositories

Public repositories on GitHub under `daniele-quero`. Source: https://github.com/daniele-quero?tab=repositories

## ac-2d-primer

- Unity/C# primer project covering 2D fundamentals, built almost entirely in C# (98%) with ShaderLab and HLSL materials.
- Includes a WebGL build and export for easy in-browser sharing.
- Link: https://github.com/daniele-quero/ac-2d-primer

## ui-game

- Unity/C# game project focused on UI systems, written entirely in C# and packaged with a WebGL build for browser play.
- Covers title screens, interactive interface elements, and UI polish.
- Link: https://github.com/daniele-quero/ui-game

## ancient-language-translator-feature

- A Unity game feature that dynamically masks in-game text based on a character's language proficiency level.
- Distributed as a ready-to-import `.unitypackage`, written in C# with custom ShaderLab and HLSL shaders.
- Link: https://github.com/daniele-quero/ancient-language-translator-feature

## enterprise-app

- Unity/C# enterprise-style application project structured for production, featuring Fakes and FakesAssemblies for unit testing patterns.
- Uses a mixed-language stack (C#, ShaderLab, HLSL) in a well-organised, team-oriented layout.
- Link: https://github.com/daniele-quero/enterprise-app

## dungeon-escape

- Unity dungeon crawling game with heavy use of custom shaders (ShaderLab 59%, HLSL 10%), featuring a shop UI and bespoke colour palettes.
- Combines C# gameplay logic with advanced visual effects built through the Unity shader pipeline.
- Link: https://github.com/daniele-quero/dungeon-escape

## 2.5D-platform

- Unity/C# 2.5D platformer featuring full player animations (idle, run, fall) and dynamic lighting baked in Unity.
- Built primarily in C# (87%) with custom ShaderLab materials for the mixed 2D/3D visual style.
- Link: https://github.com/daniele-quero/2.5D-platform

## the-great-fleece

- 3D Unity project built to explore Cinemachine and cinematography fundamentals, featuring a stealth-style scene with NPC guards and a pause menu.
- Written in C# (75%) with HLSL and ShaderLab for lighting and visual effects.
- Link: https://github.com/daniele-quero/the-great-fleece

## galaxy-shooter-2d

- 2D Unity space shooter game used as the basis for a series of development tutorials published on Medium (https://danio-quero.medium.com/).
- Features boss AI (Phase II), C# gameplay systems, and GLSL shaders for visual effects.
- Link: https://github.com/daniele-quero/galaxy-shooter-2d

## learn-c-sharp-with-unity

- A Unity-based C# learning project covering variables, methods, and custom classes through hands-on, in-engine exercises.
- Written entirely in C#, it serves as a practical complement to any beginner C# or game development course.
- Link: https://github.com/daniele-quero/learn-c-sharp-with-unity

## version-control-exercise

- A minimal Unity project created as a hands-on exercise in setting up and managing a Git repository for a Unity workflow.
- Covers the basics of `.gitignore` configuration, initial commits, and version control integration for game projects.
- Link: https://github.com/daniele-quero/version-control-exercise
